# Контентный план запуска

## Публичный старт

Репозиторий публикуется после завершения трёх материалов:

1. Быстрый старт с cURL: авторизация и проверка внешнего IP.
2. Python Requests: подключение, timeout и безопасные повторные попытки.
3. Playwright: прокси для браузерной автоматизации и регионального тестирования.

## Следующая очередь

4. `407 Proxy Authentication Required`.
5. Ротация против фиксированной сессии.
6. Региональное тестирование сайта.
7. Selenium.
8. Node.js Fetch/Axios.
9. Диагностика таймаутов и TLS.
10. Методика измерения задержки и стабильности.

## Условия выделения отдельного репозитория

Раздел становится самостоятельным репозиторием только если у него появляются:

- независимая запускаемая ценность;
- отдельный жизненный цикл версий;
- собственные issues и запросы пользователей;
- понятный владелец поддержки;
- не менее нескольких законченных материалов или рабочий инструмент.

Первый кандидат — `proxy-healthcheck`, CLI для проверки авторизации, внешнего IP, географии, задержки и ротации.

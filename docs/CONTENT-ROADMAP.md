# Контентный план запуска

## Публичный старт — готово

Первая версия включает четыре запускаемых решения:

1. Быстрый старт с cURL: авторизация и проверка внешнего IP.
2. Python/HTTPX production client: timeout, retry budget, `Retry-After` и
   безопасный машиночитаемый результат.
3. Proxy Healthcheck: success rate, latency, ротация и операционный JSON.
4. Playwright Regional Web QA: allowlist, скриншот и evidence report.

## Следующая очередь

5. `407 Proxy Authentication Required`.
6. Ротация против фиксированной сессии.
7. Node.js Fetch/Undici.
8. Selenium.
9. Диагностика таймаутов, DNS и TLS.
10. Методика измерения задержки, стабильности и стоимости успешной операции.
11. Шаблон централизованной policy/allowlist для нескольких команд.

## Условия выделения отдельного репозитория

Раздел становится самостоятельным репозиторием только если у него появляются:

- независимая запускаемая ценность;
- отдельный жизненный цикл версий;
- собственные issues и запросы пользователей;
- понятный владелец поддержки;
- не менее нескольких законченных материалов или рабочий инструмент.

Первый кандидат — `proxy-healthcheck`, CLI для проверки авторизации, внешнего IP, географии, задержки и ротации.
